Terrain pucks of any spot on Earth.

Data sources

  • Elevation: Copernicus DEM (ESA, free for personal use), NASA SRTM, JAXA AW3D, AWS Terrain Tiles. Served via OpenTopography where applicable.
  • Satellite imagery: Esri World Imagery (© Esri, Maxar, Earthstar Geographics).
  • Map / buildings / water: © OpenStreetMap contributors, via the Overpass API.
  • Place names: Nominatim reverse geocoding.
  • 3D engine: three.js.
  • Built with the help of Claude.

OpenTopography API key

Copernicus DEM is served via OpenTopography, which needs a free API key. Sign up, copy your key, and paste it here. It's stored only in your browser.

Prefer no signup? Switch the Elevation dropdown to AWS Terrain: it's keyless and global.
